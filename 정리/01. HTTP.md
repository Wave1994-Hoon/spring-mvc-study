## HTTP
#### 웹 서버
- HTTP 기반으로 동작
- 정적 리소스 제공, 기타 부가기능
    - HTML, CSS, JS, 이미지, 영상
- NGINX, APACHE

#### 웹 어플리케이션 서버
- HTTP 기반으로 동작
- 웹 서버 기능 +
- 프로그램 콛를 실행해서 애플리케이션 로직 수행
    - 동적 HTML, HTTP API
    - 서블릿, JSP, 스프링 MVC
- Tomcat, Jetty, Undertow

#### 웹 서버 vs 웹 어플리케이션 서버
- 웹 서버
    - 정적 리소스 파일
- 웹 애플리케이션 서버
    - 애플리케이션 로직
    
- 둘 경계는 사실 모호함
- 자바는 서블릿 컨테이너 기능을 제공
